How Kanz works, step by step

1
Type any place name and click Search. Kanz finds its real coordinates, pulls fresh satellite data, and runs a trained model to estimate mineral prospectivity.
2
The colored squares on the map are NOT a photo — each one is a model prediction for that patch of ground. Brighter gold/white = higher predicted probability.
3
Click any colored square to see its exact probability and coordinates. The ranked list on the left shows the strongest candidates from your last search.
4
This tool flags zones worth a closer look using free satellite data and machine learning. It does NOT replace drilling, ground surveys, or a geologist's judgment.
5
Model confidence is moderate (AUC ~0.61-0.62 on held-out global test data) — meaningfully better than guessing, but not a guarantee. Use it as a first-pass screening tool.
6
You can download the current grid as a GeoJSON file to bring into QGIS, ArcGIS, or any GIS software for deeper analysis.
