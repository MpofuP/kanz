Important:

How to read Kanz, step by step

1
This map is not a photo — it's real satellite measurements turned into color. Each pixel's color shows how strongly that spot reflects certain wavelengths of light, which can hint at mineral content.
2
Brighter/whiter areas = higher index value = stronger signal for that specific mineral indicator. Darker areas = weaker signal. This does NOT mean "definitely ore" — it means "worth a closer look."
3
The "Spectral index" dropdown changes WHICH mineral signal you're viewing: iron oxide (surface rust-like minerals), ferrous iron (a different iron mineral form), or clay (associated with certain alteration zones).
4
"Candidate" view picks out only the brightest top percentage of the map (you control how much with the slider) and highlights it — a simple way to flag zones for follow-up, not a confirmed boundary.
5
"Clusters" view groups pixels that behave similarly across all three signals at once, using an algorithm (K-means) that finds patterns without being told the answer first. Each color is a distinct group — not a confidence score.
6
Hover anywhere on the map to see the exact numeric values and real GPS coordinates for that point — useful if you want to cross-check a location against other geological data.
7
Bottom line: this tool flags where to look closer using free satellite data and simple algorithms. It cannot replace drilling, ground surveys, or a geologist's judgment — think of it as a fast, free first filter.
